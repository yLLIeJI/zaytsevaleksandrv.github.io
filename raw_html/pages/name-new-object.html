ПРЕДЛОЖИ НАЗВАНИЕ ДЛЯ БУДУЩЕГО ОБЪЕКТА ЗИМНЕЙ УНИВЕРСИАДЫ-2019

Ледовая арена для
соревнований по хоккею

Площадь объекта - 42 854 м²

Количество посадочных мест - 3500

Размер основной ледовой арены - 60×30 метров

Строящаяся ледовая арена на ул. Партизана Железняка в г. Красноярске - один из спортивных объектов Зимней универсиады-2019. Общая площадь объекта - 42 854 м², размер основной ледовой арены - 60×30 метров, объект рассчитан на 3500 посадочных мест. В период проведения Студенческих игр здесь пройдут соревнованию по хоккею.

После проведения XXIX Всемирной зимней универсиады-2019 на ледовой арене разместятся отделения спортивных школ олимпийского резерва по фигурному катанию и дзюдо. Площадка станет домашней ареной для хоккейного клуба «Сокол» и детско-юношеской школы олимпийского резерва по хоккею.
